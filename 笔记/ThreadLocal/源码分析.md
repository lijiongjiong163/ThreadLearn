使用threadLocal：

```java
ThreadLocal<Person> tl = new ThreadLocal<>();//将你想放入threadLocal的类当成泛型放进去
threadLocal.get();
```

我们来看threadLocal.get();的源码：

```java
    public void set(T value) {
        Thread t = Thread.currentThread();//获取当前线程
        ThreadLocalMap map = getMap(t);//将当前线程对象传入方法，返回一个map
        if (map != null)
            map.set(this, value);
        else
            createMap(t, value);
    }
```
